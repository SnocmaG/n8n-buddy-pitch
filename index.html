Two modes

"I know what I want" — free-text chat with smart wizard buttons.

"Help me figure it out" — AI brainstorm that explores the user's problem, refines their idea, and builds the automation.

Real-time building

Users watch their workflow being built node-by-node with streaming progress. "Setting up Gmail trigger..." "Adding Drive upload..." Zero perceived idle time.

One-click import

Signed URL — paste directly into n8n. Visual flow diagram. Step-by-step setup instructions. Sticky Note embedded in every workflow as the user's in-app manual.

Key Differentiator

The system gets smarter with every use

Auto-expanding knowledge base

When the AI encounters a node it hasn't seen before, it automatically researches the docs, creates a verified reference, and adds it to the knowledge base. Every unique generation makes future ones more accurate.

User feedback loop

"This didn't work" triggers automatic review. The fix propagates to ALL future generations. One user's bug report improves output for everyone.

Version sync

Automated sync against a live n8n instance. When n8n updates, our knowledge updates. Always current, never stale.

Sonnet = Production model

90% of Opus quality at 20% of the cost. At $0.80 avg per generation, SaaS margins are 55%. Sonnet handles the vast majority of workflows perfectly.

Opus = Admin toggle

Marginally better on complex AI agent workflows. At $4.00 avg per generation, margins go negative. Used only for A/B quality testing — not user-facing.
